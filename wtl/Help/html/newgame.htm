How to create a new game

From the main menu go to Game and then select new or from the Toolbar select the first icon. The shortcut key is Ctrl-N

A dialog appears that allows you to make all your customisations.

Piece sizes

Small a piece made up of 3 blocks
Standard a piece made up of 4 blocks
Large a piece made up of 5 blocks

Block complexity

Lowest can be placed flat on any side
Low can be placed flat on 2 sides
Medium can be placed flat on 1 sides
High can't be placed flat

Ending Level

The game will terminate when this level it is completed.

You can have a shorter game aiming only for points.

Note:

This is the only opportunity you will have to change things like grid size and which type of pieces would like to play with.
