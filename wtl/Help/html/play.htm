How to play the game

The object of the game is to arrange the pieces in such a way as to form a full row.

The more rows you can sink at a time the more points you are awarded.

Bonus points are also awarded if you drop your piece.

To start a new game press F2

Player 1 default keys

Move Left Left arrow key
Move Right Right arrow key
Rotate Piece Up arrow key
Drop Piece Down arrow key
Move Down The end key

Player 2 default keys

Move Left J key
Move Right L key
Rotate Piece I key
Drop Piece K key
Move Down The Space bar

If you want to change key settings, see Keyboard . . . in the customise dialog.
