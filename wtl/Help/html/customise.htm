How to customise a game

From the main menu go to Options and then select Customise... or from the Toolbar select the second last icon. The shortcut key is F5

You can't really customise a game once you have created it. But you may :

  • Alter piece preview
  • Alter the number of starting rows
  • Alter the starting level
  • Alter the ending level

Note:

If you want to have a game with a different grid size or different pieces create a new game.
