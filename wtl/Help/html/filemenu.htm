File menu commands

The File menu offers the following commands:

New Creates a new game.
Open Opens an existing game.
Junior StackPack Opens an easy StackPack game.
Classic StackPack Opens the classic StackPack game.
Cool StackPack Opens another StackPack variation.
StackPack Mania Opens a difficult StackPack variation.
Recent Games Opens the last 4 custom games.
Exit Exits StackPack.
