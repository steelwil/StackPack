Technical questions

Programming

  • Visual C++ .NET 2003
  • WTL (Windows Template Library) version 7.0
  • Written in about 3500 lines of code
  • The help was created using Notepad and Microsoft Help Workshop.

Functionality

StackPack does not write to the registry.

Files

StackPack only creates the following files in it's current directory

  • *.gam files for stock and custom games
  • StackPack.ini to store the current program settings
